# דוגמה תקינה של bidi isolation

המסמך מתאר את השכבות. המפתחים יצטרכו להגדיר <bdi>workflow</bdi> עם הרשאות, לשלוח אותו ל-<bdi>allow list</bdi>. <bdi>Credentials</bdi> לא יכולים להיווצר ידנית, הם חייבים להגיע מ-<bdi>External Secrets</bdi>. <bdi>workflow</bdi> שלא עומד בסטנדרטים נחסם.

הדפוס כאן עטוף כמו שצריך ולכן לא אמור לעורר את הכלל.
