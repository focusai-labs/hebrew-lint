# דוגמה לכישלון של bidi isolation

המסמך מתאר את השכבות. המפתחים יצטרכו להגדיר workflow עם הרשאות, לשלוח אותו ל-allow list. Credentials לא יכולים להיווצר ידנית, הם חייבים להגיע מ-External Secrets. workflow שלא עומד בסטנדרטים נחסם.

הדפוס הזה חייב להיעטף ב-bdi כדי שהנקודה תיראה במקום הנכון ויזואלית.
